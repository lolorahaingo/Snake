# Snake

Mini jeu Snake codé en [Processing](https://processing.org/).

## Principe

Le joueur dirige un serpent sur une grille. À chaque fruit mangé, le serpent grandit et accélère. La partie se termine si le serpent touche un bord ou se mord lui-même.

## Commandes

| Touche | Action |
|--------|--------|
| `↑` `↓` `←` `→` | Déplacer le serpent |
| `r` | Recommencer la partie |

## Lancer le jeu

1. Installer [Processing](https://processing.org/download)
2. Ouvrir `Snake.pde`
3. Cliquer sur **Run**
